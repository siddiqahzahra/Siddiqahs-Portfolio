Project Overview

The goal of this project was to allow users to easily buy items whilst abroad. Currently, Canadians are unable to use a Canadian credit card to shop online in the US at retail stores such as Amazon US, Target, and Walmart US. This app would allow users to have a tempoary subscription to big box stores.

My Contributions

As the only designer on the project, I created all the designs myself in Figma. As of right now, this app is still in development, awaiting the right mobile developer to code it!
